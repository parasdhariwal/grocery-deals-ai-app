# AI Studio
1. Create an account using your gmail on https://ai.studio/
2. generate an API key (Bottom Left Corner)
3. Keep the API key handy

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
